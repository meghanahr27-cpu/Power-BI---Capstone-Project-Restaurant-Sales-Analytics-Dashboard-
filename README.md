<h1><strong>Restaurant Sales Analytics Dashboard</strong></h1>

## 📌 Project Overview

This project presents an interactive Power BI dashboard developed to analyze restaurant sales performance across multiple cities and restaurants. The dashboard transforms raw sales data into meaningful business insights using Power Query, DAX, and data visualization techniques.

The solution enables stakeholders to monitor key performance indicators (KPIs), identify sales trends, evaluate restaurant performance, and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze restaurant sales performance across different cities.
* Monitor revenue growth and order trends.
* Evaluate customer satisfaction through ratings analysis.
* Measure delivery efficiency and operational performance.
* Enable dynamic filtering and drill-down analysis.

---
